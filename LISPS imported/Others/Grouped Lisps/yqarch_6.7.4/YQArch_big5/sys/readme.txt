軟件簡介
--------
有效率才是好工具！

「源泉建築與裝飾設計CAD工具箱(簡稱:源泉設計)」是專門針對建築設計、裝飾設計及相關專業的「輔助繪圖軟件」，它著力於為用戶打造了一個簡單、易用、快捷而強大的製圖環境。「源泉設計」的核心是由 Visual LISP 語言編寫再經編譯而成，是 AutoDesk 公司的著名計算機輔助設計(CAD)軟件 AutoCAD 的外掛程序，可運行於 AutoCAD2004-2023(32bit/64bit)/浩辰CAD8,2020-2021 等各個版本。「源泉設計」具有以下主要特點：

● 「源泉設計」是免費免註冊軟件，綠色小巧，壓縮包僅15MB多(含說明、必要圖庫和常用字庫)；
● 提供中文簡體、中文繁體、英文三個語言版本，請對應下載；
● 提供多用戶系統，你可以同時為多個公司繪圖，而擁有多個完整設置；
● 你可以用毫米、厘米、分米、米甚至是英吋來繪圖，牆厚、圖塊等設置項會相應調整；

● 功能強大而簡單易用，完全不影響原有AutoCAD或天正的繪圖習慣；
● 門窗塊完全兼容天正3,可以互為使用；

● 「源泉設計」的所有命令均組織添加在原CAD菜單的最後一項，以方便查閱；
● 通過總覽命令，可以動態設置所有命令的快捷鍵，即時生效；
● 提供豐富的「裝飾構件」、「建築構件」和「建築符號」繪圖命令，建築門、窗、牆、柱隨時隨地自動修剪；
● 提供類似 ACDSee 的圖庫管理，操作直觀、維護方便；

● 強化了 AutoCAD 的部分基本功能，如「文字處理、尺寸標注、圖層管理、編輯與繪圖」等等；
● 內置了將圖元導入Google Earth(谷歌地球)的轉換程序。



360軟件誤報說明
---------------
●由於源泉設計安裝程序 setup.exe(AutoCAD版)/浩辰gsetup.exe(浩辰版) 和卸載程序 uninst.exe(AutoCAD版)/浩辰guninst.exe(浩辰版)有涉及讀寫註冊表操作，360軟件有可能會誤報此類文件為病毒。當誤報發生時您介意或覺得不便，可以暫時刪除安裝和卸載程序，選擇手動方法安裝和卸載源泉設計。以後還需要安裝和卸載程序的用戶可到官網單獨下載:http//yqarch.cn。
●新版本的360軟件直接將源泉核心文件yqarch.vlx當作病毒，與360溝通，僅建議設置信任。
●強烈建議您使用更專業的殺毒工具，比如：火絨、金山、瑞星、卡巴斯基、諾頓等。


安裝指南
--------
安裝「源泉設計」之前建議先查殺電腦中的CAD病毒，否則有可能影響安裝進程。

將「源泉設計」軟件包解壓到你指定的任意文件夾(如:「D:\yqarch\sys」，注:路徑中不能含有括號)，點擊一鍵安裝程序setup.exe(浩辰CAD用浩辰gsetup.exe)，按提示選取需關聯的AutoCAD/浩辰CAD用戶配置文件(Profile)就可以簡單完成安裝設置過程。
如果您使用WIN10操作系統，請在AutoCAD中添加源泉的系統文件夾為受信任的位置: yqarch\sys。


AutoCAD：
● 如果一鍵方法安裝不成功，請將sys\install.vlx文件拖拽入AutoCAD繪圖區也可完成「源泉設計」安裝；
● 如果啟動文件「sys\acad.lsp」與你的自編程序衝突，安裝後可直接刪除之，此時應該將「源泉設計」啟動文件yqstart.lsp加入到AutoCAD的啟動組(用AutoCAD的ap命令)；
● 在天正建築(5.0-9.0)環境下的安裝方法與上述相同。
● 強烈推薦安裝AutoCAD擴展工具(AutoCAD express tools)。AutoCAD 2005以上本身已附帶此擴展工具，安裝AutoCAD時選擇安裝就可以了。

浩辰CAD:
● 如果一鍵方法安裝不成功可改用手動安裝，方法如下：



手動安裝源泉設計方法
--------------------
如果一鍵安裝程序未成功或者您喜歡自己動手，可以用以下方法安裝「源泉設計」(兩者無區別)：

● 將「源泉設計」軟件包完整地解壓到你指定的任意文件夾(如:「D:\yqarch\sys」，注:路徑中不能含有括號)；
● 啟動AutoCAD/浩辰CAD，運行系統配置程序「菜單(menu)\工具(Tools)\選項(Preferences)」:
  a) 在彈出的系統配置對話框中選擇「文件(Files)\支持文件搜索路徑(Support File Seach Path)」；
  b) 按【添加(Add)】按鈕加入 源泉設計 的系統路徑，如「D:\YQArch\sys」；
  c) 然後按【上移(Move Up)】按鈕將此路徑上移至其餘路徑的最上端；
  d) 按"OK"按鈕退出系統配置對話框；
● 退出重新進入AutoCAD/浩辰CAD，安裝完畢！



卸載指南
--------
退出所有AutoCAD/浩辰CAD進程後，點擊一鍵卸載程序uninst.exe(浩辰CAD用浩辰guninst.exe)，按提示選擇脫離全部或部分AutoCAD/浩辰CAD用戶配置(Profile)，以及是否刪除整個「源泉設計」安裝目錄。



手動卸載源泉設計方法
--------------------
● 在AutoCAD/浩辰CAD中可發動「yquninst」命令消除「源泉設計」與當前AutoCAD/浩辰CAD用戶配置的關聯，卸載源泉菜單等(此操作僅對當前用戶配置有效)；
● 您甚至可以什麼都不做，直接刪除「源泉設計」安裝目錄。此時再發動AutoCAD/浩辰CAD時，會提示找不到源泉菜單等，之後AutoCAD/浩辰CAD也會自動消除與「源泉設計」的關聯，也等於完全卸載。



升級指南
--------
退出所有AutoCAD/浩辰CAD進程後:

● 下載的壓縮包解壓後，將sys文件夾的內容用新版全部覆蓋(用戶文件夾user1、user2不用覆蓋)；
● 敬請保留user1、user2用戶文件夾(可以改名或增加)，這樣您的快捷鍵、用戶詞庫、圖層配置、數字行、快捷填充等等個性設置會全部保留；
● 如果修改過用戶公用圖庫資料library文件夾中的內容，也請保留相應文件；
● 發動AutoCAD/浩辰CAD後，在總覽命令(yq)中選擇「追加新版新命令」按鈕，再選擇最新的快捷鍵文件(shortcut_*.yqs)即可。

全新安裝請點擊一鍵安裝程序，詳見「安裝指南」。

    ● 修改「刷子.文字計算式及內容轉換刷子，結果寫回文字(ttR)」命令；
    ● 修改「計算輸出面積、總面積(mj)」命令，面積與文字動態關聯；
    ● 修改「計算輸出周長、總周長(zc)」命令，周長與文字動態關聯；
    ● 新添「批量文字排版對齊等(ttC)」命令；

    ● 新添「刷子.匹配圖層(ssER)」命令；
    ● 新添「刷子.匹配文字樣式(ssST)」命令；
    ● 新添「刷子.匹配文字高度(ssH)」命令；

    ● 新添「設定文字的的當前樣式(ttS)」命令；
    ● 修改「選樣式更新文字(ttU)」命令(原tts)；
    ● 新添「設定當前尺寸標注樣式(ddS)」命令；
    ● 修改「選樣式更新標注(ddU)」命令；

    ● 新添「求每個圖元矩形外框中心線或中心點(aCE)」命令；
    ● 新添「所有交點齊打斷(bA)」命令；
    ● 新添「畫線切割(bD)」命令；
    ● 修改「等分曲線(bE)」命令；
    ● 修改「引線箭頭(aab/aaf/aad)」命令；
    ● 新添「多義線增刪頂點(xxV)」命令；
    ● 新添「多段線線段的線弧轉換(xxA)」命令；
    ● 新添「修改多段線線段寬度(xxW)」命令；

    ● 新添「用戶快捷鍵設置(yqsc)」命令；
    ● 改寫及更新多個尺寸標注相關命令:
    ●   新添「直徑標注(ddD)」命令；
    ●   新添「角度標注(ddANG)」命令；
    ●   新添「修改標注精度(dd0)」命令；
    ●   修改「標注文字轉化為米(ddM)」命令；
    ●   修改「分裂尺寸標注為多個(dd2)」命令；
    ●   修改「移動標註腳點(ddE)」命令。

2008/09/02: 源泉建築 v5.8.4
    ● 推出多用戶系統，不同的用戶可以擁有各自的「快捷鍵」、「系統配置(如圖層等)」和
       個性化啟動程序等等。通過「切換當前用戶」可瞬間變化當前圖紙；
    ● 「源泉建築」升級時僅需保留「用戶文件夾」的內容，這樣就可以保留你所有的使用習慣；
    ● 「源泉建築」現在已經有一鍵安裝設置程序(setup.exe)和卸載程序(uninst.exe)，
       再不用動手設置AutoCAD搜索路徑了；
    ● 新添「切換當前用戶(yquser)」命令；
    ● 新添「卸載源泉建築(yquninst)」命令；
    ● 新添「繪製窗戶立面(wde)」命令，極大方便繪製多扇窗單元及單元組合；
    ● 從5.8.1版起僅支持AutoCAD 2000及以上版，不再支持AutoCAD R14版了。
